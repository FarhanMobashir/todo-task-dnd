# todo-dnd task

## overview

The todo-dnd task is a simple todo list application. It is written in React and uses [react-beautiful-dnd]() to render the list.

It uses the following technologies:

1. ReactJs
2. Redux-toolkit
3. React-beautiful-dnd
4. Redux Persist
5. CSS module for styling
6. React Router 6

## Guide

There is a list of static users in the database. You can use the same to login to the application.

## What user can do

1. Create new task
2. Edit task
3. Delete task
4. Move task to another list
5. User can login for a session
6. User can logout for a session
7. User can login with remember me and logout

### Pre-requisites and project setup(in local)

#### Step 1

Install [NodeJs](https://nodejs.org/en/)(if not already installed)

#### Step 2

Clone dev branch of this repo to local using:

     git clone https://github.com/FarhanMobashir/todo-task-dnd.git

#### Step 3

Go into the project folder

     cd project-name

#### Step 4

Install all the npm packages

     yarn

#### Step 5

Run the application using:

     yarn start

The application runs on **localhost:3000**
